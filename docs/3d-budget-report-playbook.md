# 青曦裝修拆解報告操作手冊

## 定位

- 用途：估價器之後的第二步 lead magnet。
- 目標：不要只讓客戶看到一個區間數字，而是讓他感覺這份內容是青曦親自整理、看得懂、可拿來做決策的初步判讀。
- 核心 CTA：邀約客戶到 `新北市汐止區` 的私宅工作室參觀，喝咖啡、體驗智能家居、看真實材質與施工細節。

- 尺寸：`A4 直式`
- 閱讀順序：`結論 -> 預算拆解 -> 條件判讀 -> 下一步`
- 視覺區塊：6 塊

- 標題：`青曦空間設計｜裝修費用拆解報告`

### 區塊 4：這份報告怎麼看

- 文案 3：`如果重視日後生活體驗，建議不要只看單一工項價格，而要看整體配置與使用情境。`

### 區塊 5：通常包含 / 不包含

- 通常包含：`基礎保護與局部拆改、水電弱電調整、木作或系統收納、油漆修飾、燈具與部分設備安裝、基礎現場管理`

## 新成屋預算分配表

### 建議拆法

- 客戶總預算先拆成：
- `工程預算 85%`
- `設計 / 監工 / 管理 10%`
- `預備金 5%`

### 工程預算經驗佔比

| 工項 | 佔比 |
| --- | ---: |
| 保護、清潔、局部拆改 | 4% |
| 泥作、防水、基礎修補 | 7% |
| 水電、弱電、迴路調整 | 13% |
| 木作工程 | 14% |
| 系統櫃、收納櫃體 | 18% |
| 油漆、批土、表面修飾 | 7% |
| 地板工程 | 8% |
| 門片、玻璃、五金 | 5% |
| 廚具、衛浴設備安裝 | 10% |
| 燈具、照明配置 | 4% |
| 空調與設備配合工程 | 6% |
| 智能家居 / 控制整合預留 | 2% |
| 現場管理、搬運、雜支 | 2% |
| **合計** | **100%** |

### 快速套用方式

- 範例：客戶總預算 `300 萬`
- 工程預算：`300 萬 x 85% = 255 萬`
- 系統櫃：`255 萬 x 18% = 45.9 萬`
- 水電弱電：`255 萬 x 13% = 33.15 萬`
